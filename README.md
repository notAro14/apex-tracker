# Apex tracker

Get player stats on Apex legends

## Apex legends API

Get your Api Key [here](https://apex.tracker.gg/site-api)

Then rename the **.env.example** file to **.env** with your API_KEY filled

```env
API_KEY=your-api-key
```

Run development

```console
yarn dev
```

Run production build

```console
yarn start
```
